# Azure-windowsserver-webhosting# Azure Windows Server Web Hosting

## Project Overview

This project demonstrates how to host a website using Microsoft Azure and Windows Server with IIS.

## Objectives

- Create an Azure Virtual Machine
- Configure Windows Server
- Install IIS
- Deploy a website
- Access the website using a public IP

## Technologies Used

- Microsoft Azure
- Windows Server

- HTML
- Remote Desktop (RDP)
